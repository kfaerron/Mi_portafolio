# Mi portafolio
## Aprendizaje
- **Organización de archivos en una página web:**  
    Aprendí a estructurar un proyecto al utilizar carpetas como `assets/` y `scripts/`, lo cual hace más ordenado y fácil de mantener el sitio.
    
- **Uso básico de HTML para contenido:**  
    Practiqué cómo agregar textos, títulos, listas e imágenes dentro de una página web. Entendí mejor cómo se construye la estructura visual de un sitio.
    Además aprendí de manera básica como se estructura el codigo en HTML. 
    
- **Publicación con GitHub Pages:**  
    Aprendí a configurar un repositorio para poder publicar el sitio en GitHub Pages, desde activar la opción de deployment hasta verificar que la página estuviera disponible.
    
- **Resolución de errores comunes:**  
    Identifique y corregí errores típicos, como problemas de rutas, direcciones de carpetas mal escritas o archivos que no se cargaban por su ubicación.
